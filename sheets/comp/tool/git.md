## 删除

删除分支：

```sh
$ git branch -d '[分支名]'
$ git branch -D '[分支名]' # 强制删除
$ git branch -r -d '[远程仓库名]/[分支名]' # 删除本地监控的远程分支
$ git push '[远程仓库名]' -d '[分支名]' # 删除远程分支
```
